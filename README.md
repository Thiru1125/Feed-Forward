# Feed-Forward 🍽️

## Overview

Feed-Forward is a Full Stack Food Redistribution Platform designed to reduce food waste and help people in need. The system connects food donors with recipients through a simple and efficient web application.

The platform allows users to donate surplus food, request food assistance, and manage food distribution effectively.

---

## Features

### User Features

* User Registration and Login
* Profile Management
* Donate Food
* Request Food
* View Available Food Donations
* Track Request Status

### Admin Features

* View Food Donation Requests
* Approve or Reject Requests
* Manage Food Listings
* Monitor Platform Activities
